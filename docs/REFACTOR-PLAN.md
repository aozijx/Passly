# Passly 重构执行手册（进度同步版）

> 目标：在不破坏现有可用性的前提下，完成一次“安全优先、可渐进落地”的结构与能力升级。

## 0. 使用方式

每次推进前，按下面顺序执行：

1. 先做 0.1 的前置核对，再开始对应 Phase。
2. 每完成一个 Phase，必须跑编译 + 关键手测。
3. 所有高风险改动都要有回滚路径（1-2 步可执行）。
4. 任何涉及 DB 的改动，必须保留向后兼容策略；备份按“当前单格式策略”执行。

### 0.1 前置核对清单

- [x] 确认当前数据库版本号（DatabaseConfig.VERSION）与线上已发布版本一致。
- [x] 确认当前备份格式版本号与实现一致（V1-Reset 格式，历史格式默认不兼容）。
- [x] 确认是否已有未提交的并行改动，避免迁移脚本冲突。
- [x] 准备至少一份真实旧数据样本用于迁移验证。

## 2. 重构目标（结果导向）

### 2.1 安全目标

- 列表与摘要模型不包含永久敏感字段（如密码明文解密结果）。
- 查看/复制/编辑敏感值必须经过认证链路。
- 禁止业务路径绕过认证直接调用“静默解密”。
- 备份采用金融级 Argon2id KDF + AES-GCM 认证加密。

### 2.2 数据目标

- 迁移链保持可回放（MIGRATION_1_2 + MIGRATION_2_3）。
- 迁移过程中不依赖 destructive migration。
- 旧数据可升级，关键字段不丢失。

### 2.3 备份目标

- 支持新备份格式（ZIP 载荷 + 加密封装 + PASSLY 幻数校验）。
- 仅支持当前备份格式（V1-Reset，历史格式不兼容，导入时明确提示）。
- 支持包含或排除媒体文件导出，全量备份自动标记 _full 后缀。

## 4. 认证再解锁（安全主线）

### 4.1 原则

- 敏感信息不预解密。
- 交互触发后认证成功才解密。
- TOTP 可作为例外路径（明确限定用途，不泛化）。

### 4.2 关键改动

1. VaultCryptoSupport
   - 删除或限制通用 decryptSilently()。
   - 仅保留认证链路解密接口。
   - TOTP 场景使用专用解密方法。
   - 当前状态：已完成。

2. showDetailForEdit() + DetailScreen
   - 不再预填解密结果。
   - 编辑页默认掩码；点击显示后走认证。
   - 当前状态：已完成。

3. VaultScreen 滑动动作
   - 所有密码复制/编辑入口统一走 ViewModel 认证解密链路。
   - 当前状态：已完成。

### 4.3 验收清单

- [x] 列表初次加载无密码解密行为。
- [x] 详情查看敏感字段时出现认证弹窗。
- [x] 认证失败时不返回明文。
- [x] 无“静默解密”绕过路径（代码搜索验证）。

## 5. 自定义图片改造（磁盘存储 + 备份打包）

### 5.1 目标状态

- 图片文件存储在应用私有目录。
- DB 仅保存相对路径（避免设备路径耦合）。
- 备份时将引用到的图片打包进 ZIP 的 images/ 目录。

### 5.2 路径策略

- 存储值：vault_images/<fileName>（相对路径）。
- 运行时通过 context.filesDir 解析绝对路径。

### 5.3 兼容策略

- 导入旧数据时支持绝对路径兜底识别。
- 启动或导入后可执行一次“路径归一化”任务。

### 5.4 验收清单

- [x] 备份文件包含 images/ 目录与 data.json。
- [x] 导出时可选择是否包含媒体文件。
- [ ] 新增图片条目后 DB 不再写入 BLOB（待 P4 收尾验证）。

## 6. 备份格式升级与可选导出

### 6.1 格式策略

- 当前版本：V1-Reset。
- 技术栈：PASSLY 幻数 + Argon2id + AES-256-GCM + ZIP。
- 兼容策略：仅接受 V1 版本头；历史版本直接拒绝。

### 6.2 导出特性

- 支持“包含媒体文件”开关。
- 文件名后缀：全量备份自动添加 _full 后缀。
- 事务性：密码错误或导出中断时自动清理 0 字节文件。

### 6.3 导入特性

- 幻数校验：准确识别非备份文件并报错。
- 密码校验：通过 AEAD Bad Tag 准确识别密码错误。

### 6.4 验收清单

- [x] V1-Reset 格式可正常导出/导入。
- [x] 历史格式（或非法文件）导入会给出准确报错。
- [x] 导出失败时磁盘无残留。
- [x] 全量与基础备份文件名生成符合规范。

## 8. 风险与回滚

| 风险     | 触发信号         | 快速止损              | 回滚策略                 |
|--------|--------------|-------------------|----------------------|
| 迁移失败   | 冷启动崩溃/迁移异常日志 | 停止发布，锁定版本         | 保留版本号，追加修复 Migration |
| 敏感字段泄露 | 列表/日志出现明文    | 立即下线相关入口          | 回滚到前一认证链路稳定版本        |
| 备份不兼容  | 历史备份导入失败      | 明确提示格式不兼容并中止导入 | 维持单格式策略，引导重新备份     |
| 图片丢失   | 导入后 icon 不显示 | 启用路径归一化修复任务       | 回退到旧路径解析逻辑           |
